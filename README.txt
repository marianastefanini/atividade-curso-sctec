Atividade Extra - Portfólio Pessoal: Mariana Stefanini

Descrição do Projeto:

Este projeto consiste no desenvolvimento de uma página de portfólio pessoal como atividade final do curso de Introdução à Programação Front-End e Back-End. A proposta foi aplicar, na prática, os conhecimentos adquiridos ao longo do curso, estruturando uma página web estática com organização visual, clareza de informações e funcionalidades básicas em JavaScript.

A página apresenta informações pessoais, habilidades técnicas e comportamentais, além de uma seção de projetos fictícios desenvolvidos com fins demonstrativos.

Objetivo:

O principal objetivo deste projeto foi consolidar os fundamentos de desenvolvimento front-end, utilizando HTML para estruturação, CSS para estilização e organização visual, e JavaScript para implementação de interações básicas.

Desenvolver uma página funcional e organizada, aplicando boas práticas de estruturação de código e separação de responsabilidades entre os arquivos.

Tecnologias Utilizadas:

- HTML5
- CSS3
- JavaScript

Projeto executado com o Visual Studio Code.

Estrutura do Projeto:

O projeto está organizado da seguinte forma:

/Atividade Final  
│  
├── index.html  
├── style.css  
├── script.js  
└── assets (imagens utilizadas no projeto)

Seções Desenvolvidas:

A página contém as seguintes seções obrigatórias:

- Nome e apresentação pessoal
- Descrição profissional
- Habilidades técnicas e comportamentais
- Lista de projetos fictícios, contendo título e breve descrição
- Formulário de contato

Funcionalidades Implementadas:

- Estruturação semântica da página com HTML
- Estilização e organização visual com CSS
- Validação básica de formulário utilizando JavaScript, impedindo o envio caso o campo obrigatório não esteja preenchido

Como Executar o Projeto:

1. Realizar o download do arquivo compactado.
2. Extrair os arquivos.
3. Abrir o arquivo `index.html` em qualquer navegador atualizado.

Não é necessário servidor.

Considerações Finais:

A atividade permitiu aplicar conceitos fundamentais de front-end e reforçou a importância da organização, clareza estrutural e funcionalidade básica em aplicações web.
